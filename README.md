Radi se o implementaciji drustvene igre Atoll koja se igra na tabli kvadratnog ili pravougaonog oblika.
Igrac je pobedio ukoliko uspesno poveze vise od (d/2)+1 ostrva, gde je d ukupan broj ostrva na tabli.
Sama igra se pokrece tako sto je prvo potrebno pozicionirati se u Game folder komandom 'cd ./Game'.
Kada se pozicioniramo u Game folder, pokretanjem main.py fajla komandom 'py ./main.py' se pokrece i sama igra.
